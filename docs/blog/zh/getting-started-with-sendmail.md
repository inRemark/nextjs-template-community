# AICoder 入门指南2

欢迎来到 AICoder！本文将带您了解如何从零开始使用我们的邮件营销平台。

## 第一步：创建账户

1. 访问我们的注册页面
2. 填写基本信息
3. 验证邮箱地址
4. 完成账户设置

## 第二步：导入客户数据

AICoder 支持多种方式导入客户数据：

- **CSV 文件上传**：批量导入联系人
- **手动添加**：逐个添加重要客户
- **API 集成**：自动同步 CRM 数据

### CSV 文件格式要求

```csv
email,name,company
john@example.com,John Doe,Tech Corp
jane@example.com,Jane Smith,Design Inc
```

## 第三步：创建邮件模板

我们提供丰富的模板库，您也可以：

1. 使用拖拽编辑器自定义模板
2. 导入现有 HTML 模板
3. 使用 AI 生成邮件内容

### 模板最佳实践

- 保持简洁明了
- 使用响应式设计
- 添加明确的 CTA（行动号召）
- 个性化内容

## 第四步：发送邮件

一切准备就绪后，您就可以开始发送邮件了！

### 发送选项

- **立即发送**：马上发送给所有收件人
- **定时发送**：选择最佳发送时间
- **A/B 测试**：测试不同版本的效果

## 开始您的营销之旅

现在您已经掌握了基础知识，可以开始创建您的第一个营销活动了。如有任何问题，请随时联系我们的支持团队。

祝您营销成功！🚀
