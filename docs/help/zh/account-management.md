# 账户管理指南

本指南将帮助您管理账户的各个方面。

## 账户设置

### 基本信息

更新您的个人信息：

- **姓名**：显示在个人资料中
- **邮箱**：用于登录和接收通知
- **电话**：选填，用于双重验证
- **头像**：支持 JPG、PNG 格式

### 修改设置

1. 进入"个人设置"页面
2. 点击"编辑信息"
3. 修改需要更改的字段
4. 点击"保存"按钮

## 安全设置

保护您的账户安全至关重要。

### 密码管理

**密码要求：**
- 至少 8 个字符
- 包含大小写字母
- 包含数字和特殊字符
- 不能与前 3 次密码相同

### 双重验证（2FA）

启用双重验证增强安全性：

1. 前往"安全设置"
2. 点击"启用双重验证"
3. 扫描二维码或输入密钥
4. 输入验证码完成设置

### 登录历史

查看最近的登录记录：

- 登录时间
- 登录设备
- IP 地址
- 地理位置

如发现异常登录，立即修改密码并联系支持团队。

## 隐私设置

### 个人资料可见性

控制谁可以看到您的信息：

- **公开**：所有人可见
- **仅团队**：仅团队成员可见
- **私密**：仅自己可见

### 数据下载

您可以随时下载您的数据：

1. 前往"隐私设置"
2. 点击"下载我的数据"
3. 等待邮件通知
4. 下载数据包

## 通知设置

自定义通知接收方式：

### 邮件通知

- 系统更新
- 账户活动
- 营销信息

### 应用内通知

- 实时提醒
- 重要消息
- 活动通知

## 账户删除

如需删除账户：

⚠️ **警告**：此操作不可恢复！

1. 前往"账户设置"
2. 点击"删除账户"
3. 确认删除原因
4. 输入密码验证
5. 点击"永久删除"

数据将在 30 天后完全删除。
