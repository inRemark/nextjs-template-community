# 常见问题排查

遇到问题？本指南将帮助您快速解决。

## 登录问题

### 无法登录

**症状**：输入正确的用户名和密码后无法登录。

**可能原因：**

1. **密码错误**
   - 检查大小写
   - 尝试密码重置

2. **账户被锁定**
   - 多次登录失败会临时锁定账户
   - 等待 15 分钟后重试

3. **浏览器问题**
   - 清除浏览器缓存和 Cookie
   - 尝试使用隐身模式
   - 更换浏览器

**解决方案：**

```bash
# 清除浏览器数据步骤
1. 打开浏览器设置
2. 找到"隐私和安全"
3. 点击"清除浏览数据"
4. 选择"Cookie 和其他网站数据"
5. 点击"清除数据"
```

### 忘记密码

使用密码重置功能：

1. 点击登录页面的"忘记密码"
2. 输入注册邮箱
3. 检查邮件收件箱
4. 点击重置链接
5. 设置新密码

💡 **提示**：重置链接 1 小时内有效。

## 页面加载问题

### 页面加载缓慢

**诊断步骤：**

1. **检查网络连接**
   - 测试网速
   - 尝试访问其他网站

2. **检查浏览器**
   - 关闭不必要的标签页
   - 禁用浏览器扩展
   - 更新到最新版本

3. **检查系统资源**
   - 关闭占用资源的程序
   - 重启电脑

### 页面显示不正常

**常见原因：**

- CSS 文件加载失败
- JavaScript 被禁用
- 浏览器兼容性问题

**解决方案：**

1. 刷新页面（Ctrl/Cmd + F5 强制刷新）
2. 启用 JavaScript
3. 使用推荐的浏览器：
   - Chrome 最新版
   - Firefox 最新版
   - Safari 最新版
   - Edge 最新版

## 功能使用问题

### 文件上传失败

**检查项：**

- ✓ 文件大小不超过 10MB
- ✓ 文件格式符合要求
- ✓ 网络连接稳定
- ✓ 浏览器支持该功能

**支持的文件格式：**

- 图片：JPG, PNG, GIF, WebP
- 文档：PDF, DOC, DOCX
- 其他：TXT, MD, CSV

### 数据未保存

**可能原因：**

1. **网络中断**
   - 检查网络连接
   - 重新提交

2. **会话过期**
   - 重新登录
   - 使用"记住我"功能

3. **权限不足**
   - 确认操作权限
   - 联系管理员

## 性能优化建议

### 提升使用体验

1. **使用最新浏览器**
   - 定期更新浏览器
   - 启用硬件加速

2. **优化网络**
   - 使用稳定的网络连接
   - 避免使用 VPN（如非必要）

3. **清理缓存**
   - 定期清理浏览器缓存
   - 关闭不使用的标签页

## 移动端问题

### 移动应用无法打开

1. 检查应用版本
2. 更新到最新版本
3. 清除应用缓存
4. 重新安装应用

### 移动网页显示异常

- 使用移动浏览器的桌面模式
- 旋转屏幕方向
- 清除浏览器数据
